Projects

Starbound

Using the Unity Engine and C#, I developed a top-down 2D bullethell game named Starbound. Visual and Auditory assets were either repurposed from assets found in game asset stores or created in Aesprite/Audacity. The project demonstrates use of Data Structures, Algorithms and OOP Class Structure. Multivaritate algebra and geometry skills were needed to create interesting enemy patterns. Coroutines were used to implement multitasking and complex sequences.

Charting Module

I created a charting module for a desktop application created in Microsoft's Windows Forms framework using C#. The application would take data in the form of a CSV file and prompt the user to select various parameters of varying data types. The application plots the provided data with axes that adjusted to the ideal range with custom zoom functionality. I implemented multichart functionality, which allowed the user to overlay charts, adjusting axes and units automatically.

Personal Website

A website I made to learn the fundamentals of CSS/HTML. In particular, I used this project to get familiar with CSS's Box Model and Flexbox framework. It showcases some of my projects, essentially acting as a resume in website form. Themes, design, and colors were taken from The Odin Project's module on fundamentals. Images used are stock images or images sourced by me.

Menuing System

For a school project, I designed a working menuing system in Java. It allowed for users to change various visual aspects of a user configuration, auditory levels of sound effects and background music, as well as name the configuration. The application saves/loads the desired settings in JSON format and loads user configurations as desired. The project made use of Unit Testing to create robust methods and classes.

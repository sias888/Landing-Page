About

Projects

Using the Unity Engine and C#, I developed a top-down 2D bullethell game named Starbound. Visual and Auditory assets were either repurposed from assets found in game asset stores or created in Aesprite/Audacity. The project demonstrates proper use of Data Structures, Algorithms and OOP-based Class Structure. Multivaritate algebra and geometry skills were needed to create interesting enemy bullet patterns. Coroutines were used to implement multitasking and complex sequences.

I created a charting module for a desktop application created in Microsoft's Windows Forms framework using C#. The application would take data in the form of a multidimensional array and prompt the user to select various parameters of varying data types. The application plots the provided data with axes that adjusted to the ideal range with custom zoom functionality. I implemented multichart functionality, which allowed the user to overlay charts, adjusting axes and units automatically.
